# Engineering Workflow

## Engineering Lifecycle
1. **Feature Request**
2. **Engineering CLI**: `doctor`, `context`, `prompt`
3. **Gemini CLI**: Implementation
4. **Technical Lead Review**: Handover Package
5. **Validation**
6. **Git Commit**
7. **Merge**

## Responsibilities
- **Implementation AI**: Analyzes, implements, tests, and packages.
- **Technical Lead**: Validates architecture and implementation quality.
- **Human Developer**: Approves and commits changes.

## Engineering Principles
- Simple solutions, incremental changes, reusable components.

## Scope Guardrails
- Strictly enforce task scope. No unrelated refactoring.
